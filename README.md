<--- 개요 --->

이 프로젝트는 개인 프로젝트로 진행한 3D 액션게임 "Project : Combo" 입니다.

게임 '명조 : 워더링웨이브', '마비노기 영웅전'에 모티브를 얻어 제작한 게임으로,
일반공격 & 스메쉬 시스템, 장비, 인벤토리, 쇼트컷, 스킬트리, 스테이지, 세이브 & 로드 시스템을 구현한 프로젝트입니다.
